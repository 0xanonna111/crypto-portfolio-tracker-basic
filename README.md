# Crypto Portfolio Tracker Basic

This project helps beginners learn how to build a simple crypto portfolio tracker using Web3 tools.

You will learn:
- How to connect a wallet
- How to read wallet balances
- How to fetch token prices
- How to show data on a webpage
- How crypto dashboards work

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/crypto-portfolio-tracker-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- portfolio.js = Portfolio functions
- web3.js = Blockchain connection
- api.js = Price API handler
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Show ETH balance
- Show token prices
- Show total value
- Simple dashboard

## Learning Goal
This project explains how crypto wallets and price data can work together.
